# E-Commerce Sales Analytics Dashboard

## Project Overview
This project is an interactive Business Intelligence dashboard built using Power BI and MySQL to analyze e-commerce sales performance, profitability, customer behavior, and regional trends.

## Tools & Technologies
- Power BI
- MySQL
- DAX
- SQL

## Key Features
- Executive KPI Cards
- Drill-through analysis pages
- Dynamic slicers and filters
- Time intelligence calculations
- Customer and regional performance analysis
- Conditional formatting for KPI indicators
